# taro-react-starter

一个开箱即用的基于`Taro3` + `React` + `Zustand` + `TailwindCSS` + `TypeScript`的项目模板

## 使用

```sh
pnpm install
```

### 运行

```sh
pnpm dev:h5
pnpm dev:weapp
```
